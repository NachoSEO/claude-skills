# Claude Skills

A collection of custom Claude Code skills/plugins.

## Skills

| Skill | Description |
|-------|-------------|
| [google-search-engineer](./google-search-engineer/) | SEO advisor grounded in leaked Google ranking signals from the Content Warehouse API |

## Installation

Install any skill as a Claude Code plugin:

```bash
claude plugins add /path/to/claude-skills/<skill-name>
```

For example:

```bash
claude plugins add /path/to/claude-skills/google-search-engineer
```
